# المرحلة السابعة: التقارير

## SQL

شغّل الملف التالي بعد ملفات المراحل السابقة:

```txt
supabase/migrations/0006_reports_read_access.sql
```

هذا الملف يمنح Viewer صلاحية قراءة بيانات التقارير فقط، مع بقاء صفحات التشغيل مثل المهام والمشاريع محجوبة عنه من التطبيق.

## الرابط

```txt
http://localhost:3000/reports
```

## ما يجب تجربته

- تقرير يومي.
- تقرير أسبوعي.
- فلترة حسب المستخدم.
- فلترة حسب المشروع.
- مراجعة إحصائيات المهام.
- مراجعة إحصائيات الإنجازات.
- مراجعة إحصائيات الدورات.
- الضغط على زر `طباعة التقرير` واستخدام طباعة المتصفح أو Save as PDF من نافذة الطباعة.

## الصلاحيات

- Admin و Team Lead و Member و Viewer يستطيعون دخول `/reports`.
- Viewer يقرأ التقرير فقط ولا يستطيع دخول صفحات التشغيل الأخرى.
