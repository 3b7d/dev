# المرحلة الخامسة: المشاريع

## SQL

شغّل الملف التالي بعد ملفات المراحل السابقة:

```txt
supabase/migrations/0004_projects.sql
```

## الرابط

```txt
http://localhost:3000/projects
```

## الاختبار حسب الدور

- Admin و Team Lead: إضافة مشروع، تعديل مشروع، حذف مشروع، إضافة روابط، حذف روابط، إضافة أحداث Timeline، حذف أحداث Timeline.
- Member: مشاهدة المشاريع والمهام المرتبطة فقط.
- Viewer: لا يستطيع دخول `/projects` ويتم توجيهه إلى `/unauthorized`.

## ما يجب تجربته

- إضافة مشروع جديد بحالة `نشط`.
- تحديد مالك المشروع.
- ضبط نسبة الإنجاز من 0 إلى 100.
- إضافة رابط للمشروع.
- إضافة حدث في Timeline.
- إنشاء مهمة مرتبطة بالمشروع من قسم المهام.
- الرجوع إلى Dashboard ومراجعة ظهور المشروع ضمن المشاريع النشطة.
