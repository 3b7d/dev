# المرحلة الرابعة: إنجازات اليوم

## SQL

شغّل الملف التالي بعد ملفات المراحل السابقة:

```txt
supabase/migrations/0003_daily_achievements.sql
```

## الرابط

```txt
http://localhost:3000/achievements
```

## الاختبار حسب الدور

- Admin و Team Lead: إضافة إنجاز لأي مستخدم، ربطه بمهمة مكتملة، وتعديل/حذف الإنجازات.
- Member: إضافة إنجاز لنفسه فقط، وربطه بمهامه المكتملة المتاحة له.
- Viewer: لا يستطيع دخول `/achievements` ويتم توجيهه إلى `/unauthorized`.

## ما يجب تجربته

- إضافة إنجاز بتاريخ اليوم.
- ربط الإنجاز بمهمة مكتملة.
- كتابة خطة الغد.
- كتابة عائق.
- فلترة النتائج حسب التاريخ والمستخدم.
- مراجعة التقرير اليومي البسيط.
