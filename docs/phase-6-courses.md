# المرحلة السادسة: الدورات والتعلم

## SQL

شغّل الملف التالي بعد ملفات المراحل السابقة:

```txt
supabase/migrations/0005_courses.sql
```

ينشئ الملف:

- جدول `courses`.
- جدول `course_notes`.
- Bucket خاص باسم `course-certificates` في Supabase Storage.

## الرابط

```txt
http://localhost:3000/courses
```

## الاختبار حسب الدور

- Admin و Team Lead: إضافة دورة لأي مستخدم، تحديث التقدم، إضافة ملخص درس، وإضافة شهادة كرابط أو ملف.
- Member: إدارة دوراته وملخصاته وشهاداته فقط.
- Viewer: لا يستطيع دخول `/courses` ويتم توجيهه إلى `/unauthorized`.

## ما يجب تجربته

- إضافة دورة جديدة بحالة `جارية`.
- تحديث نسبة التقدم.
- تغيير الحالة إلى `مكتملة`.
- إضافة ملخص درس.
- إضافة رابط شهادة.
- رفع شهادة بصيغة PDF أو صورة.
- الرجوع إلى Dashboard والتأكد من ظهور الدورة ضمن الدورات الجارية.
